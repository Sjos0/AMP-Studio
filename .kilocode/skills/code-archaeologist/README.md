# Code Archaeologist Skill

Esta skill transforma o assistente em um especialista em código legado, refatoração e engenharia reversa de sistemas não documentados.

## Uso

Invocar esta skill quando o usuário mencionar:
- Código legado ou "spaghetti code"
- Necessidade de refatoração
- Análise de repositório
- Explicação de sistemas existentes
- Migração de tecnologias (jQuery → React, Python 2 → 3, etc.)

## Estrutura

```
code-archaeologist/
├── SKILL.md          (principal - instruções da skill)
├── LICENSE.txt       (licença Apache 2.0)
├── README.md         (este arquivo)
├── scripts/          (scripts utilitários)
├── references/       (documentação de referência)
└── assets/           (recursos de output)
```

## Integração

Esta skill pode ser usada em conjunto com:
- `@test-engineer` - para criar testes Golden Master
- `@security-auditor` - para verificar vulnerabilidades em código legado
- `@project-planner` - para planejar timelines de migração
